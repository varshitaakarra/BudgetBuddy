BudgetBuddy is a personal finance web application designed for college students to help track and categorize expenses automatically using OCR and AI-powered receipt analysis.
## Features
- Upload receipt images directly into the application
- Extract receipt text using OCR technology
- Automatically categorize purchases using the Gemini API
- Responsive user interface designed in Figma
- Expense categories such as groceries, clothing, dining, etc.
- Clean and intuitive budgeting dashboard
- A pie chart the summarizes the amount of money spent out of the user's total budget

 ## Tech Stack

Frontend:
- HTML
- CSS
- JavaScript

Design:
- Figma

APIs & Tools:
- OCR API / OCR.js
- Gemini API

## How It Works

1. The user uploads a receipt image.
2. OCR technology extracts text from the receipt.
3. The extracted text is sent to the Gemini API.
4. Gemini analyzes the receipt items and categorizes them automatically.
5. The categorized expenses are displayed in the application dashboard.
